- Bilingual FR/EN — FR is the default locale.
- The portfolio leans into a **magazine / sketchbook editorial** voice: numbered sections (`№01`, `№02`…), serif italics for accent words (Fraunces italic), uppercase mono micro-text (JetBrains Mono), and handwritten Caveat for hints/captions ("drag-moi !", "déplace-les ↘", "tu peux tout réorganiser").

Other available palettes: `cream`, `noir` (dark mode), `forest`, `ink` (alt dark) — see the original handoff if a darker CV variant is needed.

### Typography
| Family | Use |
|---|---|
| **Fraunces** (serif, 400 / 500 / 600 / 700, italic) | Display headings, body in editorial sections, big numbers, accent italic words |
| **JetBrains Mono** | Section meta (№01, dates, kicker labels, issue bar, tags, micro-text) |
| **Caveat** (cursive) | Handwritten hints, polaroid captions |
| Inter / system-ui | Fallback body |

### Spacing / structure
- Section vertical rhythm: `padding: 100px 60px`
- 1px solid `--ink` separator between rows / sections
- Cards: 4px radius + flat 3px shadow `3px 3px 0 var(--ink)` (sketchbook vibe — no blur)
- Polaroids: white card, 1px ink border, 6px×8px shadow with 20px blur, no radius
- Pills / buttons: `border-radius: 999px`, 1.5px ink border
- Marquees: full-bleed black band (`--ink` bg, `--bg` text), italic Fraunces 22px

### Iconography
- Squiggle (wavy underline) — used as accent under active nav items, behind names
- Hand-drawn arrow (curved + chevron head) — for hints, CTA "↗"
